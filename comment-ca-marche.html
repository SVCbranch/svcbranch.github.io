Comment ça marche ?

  1. Rendez-vous dans un bureau de tabac ou un point de vente agréé.
  2. Demandez une recharge ItPCS du montant souhaité.
  3. Payez votre recharge en espèces ou par carte bancaire.
  4. Le distributeur vous remettra un ticket contenant un code unique.
  5. Utilisez ce code sur les sites partenaires ItPCS pour créditer votre compte.

FAQ

Qu’est-ce qu’une carte ItPCS ?

Il s’agit d’une carte de paiement prépayée, que vous pouvez recharger à tout moment. Elle fonctionne aussi bien en ligne qu’en boutique, sans être liée à un compte bancaire. Parfait pour garder le contrôle sur vos dépenses.

Où acheter une recharge ItPCS ?

Rendez-vous dans un bureau de tabac ou un point de vente agréé (comme une station-service ou un magasin de presse) pour acheter votre recharge ItPCS.

Comment recharger ma carte ?

Achetez une recharge ItPCS chez un revendeur. Vous recevrez un ticket avec un code unique, à entrer ensuite sur notre site ou chez nos partenaires.

Peut-on acheter une recharge en ligne ?

Non. Les recharges ItPCS sont uniquement disponibles en point de vente physique. Aucun achat en ligne n’est proposé actuellement.

Mon code ne fonctionne pas. Que faire ?

Vérifiez que vous avez bien saisi le code. En cas d’erreur persistante, contactez notre service client avec votre ticket et le code concerné.

Carte vs recharge : quelle différence ?

La carte ItPCS est le support que vous utilisez pour vos paiements. La recharge, elle, permet d’ajouter du crédit sur cette carte. Les deux sont complémentaires.

Y a-t-il une date d’expiration ?

Oui, chaque recharge a une date de validité. Vous la trouverez indiquée directement sur le ticket remis en point de vente.
