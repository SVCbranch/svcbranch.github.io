Comment ça marche ?

  1. Rendez-vous dans un bureau de tabac ou un point de vente agréé.
  2. Demandez une recharge ItPCS du montant souhaité.
  3. Payez votre recharge en espèces ou par carte bancaire.
  4. Le distributeur vous remettra un ticket contenant un code unique.
  5. Utilisez ce code sur les sites partenaires ItPCS pour créditer votre compte.

FAQ

Qu'est-ce qu'une carte ItPCS ?

Une carte ItPCS est une carte de paiement prépayée rechargeable, utilisable en ligne et dans les magasins physiques. Elle n'est liée à aucun compte bancaire et permet de contrôler ses dépenses.

Où puis-je acheter une recharge ItPCS ?

Vous pouvez acheter une recharge ItPCS dans la plupart des bureaux de tabac et dans certains autres points de vente agréés (magasins de presse, stations-service...).

Comment recharger ma carte ItPCS ?

Pour recharger votre carte ItPCS, vous devez acheter une recharge (un ticket) dans un point de vente physique. Ce ticket contient un code unique que vous utiliserez ensuite sur le site de ItPCS ou sur les sites partenaires.

Puis-je acheter une recharge ItPCS en ligne ?

Non, ItPCS ne propose pas la vente de recharges ItPCS en ligne. Les recharges ItPCS s'achètent exclusivement en points de vente physiques.

Mon code de recharge ne fonctionne pas. Que faire ?

Vérifiez d'abord que vous avez correctement saisi le code. Si le problème persiste, contactez le service client de ItPCS, munissez de votre ticket de caisse et du code de recharge.

Quelle est la différence entre une carte ItPCS et une recharge ItPCS ?

La carte ItPCS est le support physique (la carte elle-même). La recharge ItPCS est le moyen de créditer cette carte. Vous achetez une recharge pour ajouter de l'argent sur votre carte ItPCS.

Les recharges ItPCS ont-elles une date d'expiration ?

Oui, les recharges ItPCS ont généralement une date d'expiration. Vérifiez la date indiquée sur votre ticket.
